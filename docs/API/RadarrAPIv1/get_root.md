## Summary

Queries the status of root folder.

## Parameters

Required: None

Optional: None

## Example

```python
get_root()
```

## Returns JsonArray

```json
[
    {
        'path': '/movies/',
        'freeSpace': 3851519787008,
        'totalSpace': 4710500007936,
        'id': 8
    }
]
```
